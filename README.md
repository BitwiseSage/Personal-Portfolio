\# Arfan Asgar — Developer Portfolio



Personal portfolio website built using React, TypeScript and TailwindCSS.



\## Tech Stack

\- React

\- TypeScript

\- TailwindCSS

\- Vite

\- Framer Motion



\## Features

\- Responsive design

\- Animated UI components

\- Project showcase

\- Skills section

\- GitHub and LinkedIn integration



\## Setup



Install dependencies:



npm install



Run development server:



npm run dev
